#  제어구문 실습



### 다중 if, if문 중첩



Scanner클래스를 이용해서 입력받도록

0부터 100까지 입력되면 학점평가, 외의 숫자는 잘못입력으로 출력 되도록 구현

0 ~ 59 : F학점

60~69 : D학점

70 ~ 79 : C학점

80 ~ 89 : B학점

90 ~ 100 : A학점

110 or -40 : 잘못입력

[출력형태]

점수 : xxx

xxxx학점
